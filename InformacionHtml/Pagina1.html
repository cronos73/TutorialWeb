BÁSICO HTML

¿Qué es?

El Lenguaje de Marcado de Hipertexto (HTML) es el código que se utiliza para estructurar y desplegar una página web y sus contenidos.

Estiquetas


La estructura HTML de una página web se compone de las etiquetas :

  • Html:Indica el comienzo de la página
  • Head:Contiene información descriptiva
  • Body:Conforma todos los elementos visibles

Existen dos tipos de etiquetas HTML:

  • Etiqueta Emparejada:Vienen en pares y tienen una apertura y cierre.
  • Etiqueta y Cierre:Se refiere a las etiquetas que no requieren cierre.

Estructura


  • !DOCTYPE html:se utiliza para indicar la versión HTML 5.0 (la usada actualmente).
  • Html:Es el elemento raíz y es el más importante dentro de la estructura HTML, pues es el que contendrá el resto de las etiquetas de atributos de la página web.
  • Head:es la etiqueta principal que incluye metadatos; son atributos que no se muestran al usuario, solo describen referencias de la página, por ejemplo, título o página CSS.
