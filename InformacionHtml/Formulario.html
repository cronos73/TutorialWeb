Formularios

Etiquetas 

      Form:Se utiliza para recopilar información
      Input:Es el más importante muestra de varias maneras el atributo
      Botón submit:Para enviar los datos
      action:Define la acción de realizar se envía el Formulario
      target: Especifica si el formulario se enviara a una nueva página el resultado 
      method:Este atributo especifica el metodo HTTP que puede ser GET O POST se usa para cuando se envíe los datos .
      name:Este es paraque los datos se manden 
      fieldest:Agrupa los datos 
      label:Lo que ve el usuario

Autocompletado desactivado

En esta si se pone en un campo no se vera escrita la información anerior del campo

Autocompletado activado

En esta si guarda la información del anterior campo

Placeholder

Este es el texto que aparece como sugerencia en el campo de escribir y cuando empieze a escribir desaparecera

Value

Este funciona para que se quede un dato establecido y no se pueda cambiar

Size

Esta funciona para que puedas poner el tamaño del donde se escribe lo hace grande

Input extensos

Method GET

    Este método por defecto  seran visibles los datos en la URL de la página de envío
    *La longitud de una URL es Limitada(2048 caracteres) 
    *NOTA:No usar get para datos confidenciales 
    *Es mejor para buscadores

Method POST

Siempre hay datos confidenciales se debe usar POST .Este método no muestra la información en la URL y no tiene limitación de tamaño

Tipos de Botones


Botón Normal

El primer botón sirve para enviar los formularios una vez que el usuario los rellene


Botón Imagen

Muestra una imagen como boron donde se puede poner el tamalo etc


Botón Reset(Borrar)

Funciona para que borre todos los valores por defecto del formulario si deseas iniciar de nuevo le das clic hay y los borra


Botón alternativo

Botones alternativos son aquellos que por defecto se enviaran distinto


Botón Personalizado

Este botón es para utilizarlo en javascript ya que no tiene alguna funcialidad y tu se la pones
